## Project Overview

Wi-Fi (WLAN) packet manipulation using Scapy. This project deals with crafting, sending, sniffing, and analyzing 802.11 frames programmatically in Python.

## Development Environment

No build/lint/test tooling has been configured yet. Common commands to set up:

```bash
# Create a virtual environment
python3 -m venv venv
source venv/bin/activate

# Install Scapy
pip install scapy
```
